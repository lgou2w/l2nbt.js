A lgou2w Minecraft NBT library for Javascript

## Install

```
npm install l2nbt --save
```

or

```
yarn add l2nbt
```

There is no default export. The correct way to import `l2nbt.js` is:

```javascript
import * as l2nbt from 'l2nbt'
```
